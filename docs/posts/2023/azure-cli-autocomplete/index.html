How to enable azure cli autocomplete in macos zsh shell#

Azure-cli comes with an autocomplete script that should be located in

/usr/local/etc/bash_completion.d*´

directory. The particular scritp is named ‘az’

For enabling autocomplite you just have to source that script:

source /usr/local/etc/bash_completion.d/az

Enable azure cli autocomletion by default#

Add the line zo you .zshrc file:

echo "source /usr/local/etc/bash_completion.d/az" >> ~/.szhrc

If you encounter an error when restarting, you may have to add that line before the script loading to ensure cross compatibility between bash and zsh script

autoload -U +X bashcompinit && bashcompinit

To relaod .zshrc, simlpy source it:

source ~/.zshrc

That’s it!
